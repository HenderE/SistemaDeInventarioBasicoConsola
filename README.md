## Descripción
Se requiere desarrollar una aplicación básica en consola que permita gestionar productos de forma simple, registrar información del inventario y estadísticas básicas del negocio. Este repositorio busca ofrecer una primera versión funcional del sistema de inventario que pueda ejecutarse en la terminal y que sirva para futuras mejoras del sistema.
